XCORE Math Library
==================

Introduction
************

.. toctree::

``lib_xcore_math`` is a library of optimized math functions for taking advantage of the vector
processing unit (VPU) on the XMOS XS3 architecture. Included are functions for block floating-point 
arithmetic, fast Fourier transforms, linear algebra, discrete cosine transforms, linear filtering 
and more.

See :ref:`getting_started` to get going.

On GitHub
---------

Get the latest version of ``lib_xcore_math`` at https://github.com/xmos/lib_xs3_math.
